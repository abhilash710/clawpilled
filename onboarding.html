Getting Started — Your First Week

Read this before the setup call. 5 minutes.

The One Thing to Understand

You're onboarding a teammate, not installing software.

This agent wakes up with no memory every session. The files in this workspace ARE its brain. The more you invest in teaching it who you are and what you care about, the better it gets.

The first few days will feel like onboarding a new hire. Lean into that feeling. It's exactly right.

The Mindset Shift

"Building the machine that builds machines"

Don't think of your agent as a tool you USE. Think of it as a machine you're BUILDING. Every conversation makes it smarter. Every correction becomes a rule. Every file you write gives it more context.

The meta-learning loop: build → hit a wall → figure it out → write it down → apply it next time. This loop is the entire game.

"Don't do knowledge work. Get agents to do it. If you can't, figure out how."

Every time you're about to do something manually — research, writing, analysis — pause and ask: "Can my agent do this?" If yes, delegate. If no, figure out what's missing and add it. Over time, the set of things only YOU can do shrinks to judgment, taste, and decisions.

The Setup (with Abhi)

Hour 1: Install

Install OpenClaw, connect your first channel, get the agent talking.

Hour 2: Make It Yours

Name your agent. Give it a vibe. This changes how you relate to it.

Tell it about you. Don't type — dictate. Talk about what you're dealing with. Your voice is faster than your fingers and more honest.

Have your first real conversation. Not "what's the weather" — tell it what you're struggling with. Let it push back.

Days 1-3: Build the Habit

  • Start each day by talking to your agent. Or run /where-am-i.
  • When you make a decision, tell it. When you change your mind, tell it.
  • When it gets something wrong, correct it. These corrections are the most important input. Each becomes a rule.
  • When it pushes back, don't dismiss it. Pushback = thinking partner, not broken tool.

The Compound Curve

Day 1: Your agent knows nothing. It's generic and sometimes wrong.

Day 7: It knows your world. Conversations start where yesterday left off.

Day 30: It anticipates what you need. It catches your patterns before you do.

Day 90: It's entirely shaped to how YOU think.

Be patient with the early days. Be direct with corrections. The investment compounds faster than you expect.

When something breaks, text Abhi. That's what the first few weeks are for.
